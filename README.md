# Weekly Planner

Aplicação web para organização de tarefas semanais, utilizando planejamento, Kanban e relatórios de produtividade.

O Weekly Planner foi desenvolvido com o objetivo de praticar conceitos básicos de desenvolvimento web, como estruturação de páginas, manipulação do DOM com JavaScript e organização de múltiplas telas.

O sistema permite que o usuário cadastre tarefas, organize sua semana, acompanhe o andamento pelo Kanban e visualize relatórios de desempenho ao final do período. 


# Como executar o projeto? 

1. Clone o repositório
2. Abra o arquivo `telaCadastro.html` no navegador

Ou apenas abra o arquivo `telaCadastro.html` no navegador. 


# Funcionalidades

- Cadastro de tarefas
- Planejamento semanal
- Quadro Kanban (À fazer, Fazendo e Feito)
- Relatório de progresso
- Armazenamento em LocalStorage 

# Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage
